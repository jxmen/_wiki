  • CPU Burst란?
    • '프로세스'가 CPU 작업이 계속해서 일어나는 시간
  • I/O Burst란?
    • '프로세스'가 I/O 작업이 끝날 때 까지 대기하는 시간

일반적으로는 CPU는 짧게 쓰고 끝나는 편. (표 참조)
